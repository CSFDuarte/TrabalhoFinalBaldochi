Sobre o site
  • Site desenvolvido pelos alunos: Guilherme Bastos, Gabriel Piffer e Caique Duarte para a disciplina COM222 - Desenvolvimento Web - Instituto de Matematica e Computação - Universidade Federal de Itajubá - MG - Brasil - 2019.2
  • As ferramentas utilizadas para desenvolvimento foram:
    1. Framework Angular para o front-end
    2. Interpretador javascript Node.js no back-end
    3. Banco de dados relacional Mysql
    4. Xampp para upar o servidor
  • Todo conteúdo da página é gerado Realtime, não se fazendo necessário o uso de URL
  • Este site consiste em um E-commerce de uma livraria, com ele o usuário pode navegar pelas seguintes funcionalidades:
    1. Cadastrar um usuário
    2. Buscar livros por suas categorias
    3. Buscar livros por meio de pesquisa
    4. Visualizar informações dos livros
    5. Fazer a compra de livros via Carrinho de Compra
    6. Visualizar histórico de compra
    7. Acessar informações sobre o site
